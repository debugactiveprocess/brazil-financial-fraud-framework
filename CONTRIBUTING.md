# Contribuindo para a Matriz Brasileira de Fraudes Financeiras

Obrigado pelo interesse em contribuir.

Este projeto busca documentar comportamentos observáveis de fraudes financeiras no Brasil de forma estruturada, defensiva e útil para prevenção, detecção, resposta e investigação.

## Como contribuir

Você pode abrir uma issue ou pull request para:

- sugerir nova técnica;
- corrigir descrição, detecção ou mitigação;
- adicionar exemplos brasileiros;
- melhorar mapeamentos com F3/MITRE ATT&CK;
- melhorar a interface, dados ou documentação.

## Critérios para novas técnicas

Uma técnica deve preferencialmente:

- representar uma ação observável do fraudador;
- ser útil para defesa, investigação ou inteligência;
- ter relação com fraude financeira no Brasil;
- não depender de detalhes excessivamente específicos de uma única instituição;
- incluir sinais observáveis, detecção, mitigação e evidências úteis quando possível.

## Aviso de segurança

Não envie dados pessoais, segredos, credenciais, chaves PIX reais, documentos, números de conta, dados bancários, informações sigilosas de clientes ou evidências sensíveis sem anonimização.

## Independência

Este projeto é independente e não é afiliado, endossado ou mantido pela MITRE, pelo Center for Threat-Informed Defense ou pelo projeto Fight Fraud Framework.
